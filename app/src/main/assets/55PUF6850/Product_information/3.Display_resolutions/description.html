计算机格式

  • 分辨率 - 刷新率:
    • 640 x 480 - 60Hz
    • 800 x 600 - 60Hz
    • 1024 x 768 - 60Hz
    • 1280 x 1024 - 60Hz
    • 1920 x 1080 - R 60Hz （限FULL HD，R：减少消隐时间）

视频格式

  • 分辨率 - 刷新率:
    • 480i - 60Hz
    • 480p - 60Hz
    • 576i - 50Hz
    • 576p - 50Hz
    • 720p - 50Hz, 60Hz
    • 1080i - 50Hz, 60Hz
    • 1080p - 24Hz, 25Hz, 30Hz, 50Hz, 60Hz.

注

当TV 遇到不支持分辨率和刷新率, 会有黑屏或花屏状况发生, 请切换到TV 能显示分辨率及刷新率使用。
